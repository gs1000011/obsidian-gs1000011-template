# すべてのノートにあるプロパティ

基本的にすべてのノートにつけているプロパティです。ただし `01_Log` に記録されるデイリーノートだけは `created_at` のみ付けています。

| プロパティ名 | 説明 |
| ------------ | ---- |
| tags         |   Obsidian 標準のプロパティ。本来はノートの分類をするためのもの。このノート術では興味を持ったキーワードを可視化するために使用しています。   |
| reference    |   書き切ったノートかを確認するプロパティ。今後作成するノートではココにチェックがあるノートだけリンクさせます。   |
| label        |   ノートを分類するためのプロパティ。本来 tags で管理するべきであったものを代わりに記載します。主に Bases でのクエリに使用しています。   |
| created_at   |  ノートの作成日時を記録するプロパティ。templater プラグインの機能で自動で入力されます。    |

# テンプレートに付けているプロパティ

## 03_Topic

| プロパティ名 | 説明                                                                                                                    |
| ------------ | ----------------------------------------------------------------------------------------------------------------------- |
| eye_catch    | [03_Topic](04_View/03_Topic.base) で表示するアイキャッチ画像を指定するプロパティ。                                      |
| aliases      | Obsidian 標準のプロパティ。ノートのタイトル以外の名称を記述すると、検索時にその名称でもサジェストされるようになります。 |
| reading      | 単語の読みを記入するプロパティ。難しい単語があった際に読み方を記録しています。不要であれば削除してください。        |

# 02_Event に保存される Full Calendar で作成したノートのプロパティ
